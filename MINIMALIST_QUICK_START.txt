WHAT IS MINIMALIST DESIGN?
================================================================================

Reducing text by 60-73% while IMPROVING clarity and scannability.

================================================================================
KEY PRINCIPLES (5 CORE RULES)
================================================================================

1. ONE EMOJI PER SECTION
   ✗ 🔥 Hot ✨ Deal 💥 Now ⚡ Buy     (emoji overload)
   ✓ 🔥 Hot deals                   (single anchor)

2. SINGLE-LINE LISTS
   ✗ 1. Product                      (4 lines per item)
     $25
     Stock: 5
   ✓ 1. Product — $25 | 5 шт        (1 line per item)

3. NO REDUNDANT LABELS
   ✗ Магазин: fobos (separate line)
   ✓ 📦 Товары • fobos              (inline)

4. INLINE METADATA
   Use separators: • (bullet) — (dash) | (pipe)
   Example: 1. iPhone — $999 | ✅ In stock

5. MOBILE REALITY
   Max 40 characters per line (iPhone SE = 375px width)
   If it wraps horizontally → too long

3 CONCRETE CODE CHANGES
================================================================================

CHANGE 1: Remove Label Redundancy
──────────────────────────────────
OLD:  `Магазин: ${shopName}`           (separate line)
NEW:  `📦 Товары (5) • ${shopName}`     (inline)

CHANGE 2: Compact Headers
──────────────────────────
OLD:  let msg = `📦 Мои товары (${count}):\n`;
      msg += `Магазин: ${shopName}\n\n`;

NEW:  let msg = `📦 Товары (${count}) • ${shopName}\n`;

CHANGE 3: Smart Stock Status
─────────────────────────────
OLD:  `   ⚠️ Запас: ${stock}\n`

NEW:  Create helper:
      const status = stock === 0 ? 'нет' : 
                     stock <= 3 ? `${stock} шт` : 'есть';
      Then use: `| ${status}`

Result: "1. Holo — $25 | нет" (one line!)
